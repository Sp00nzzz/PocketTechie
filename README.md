# Pocket Techie

Pocket Techie is a playful virtual desk pet that impersonates your favorite hustle-hungry tech bro. Feed him shawarma, poke the retro UI buttons, and watch him spit out motivational dev chatter.

## Features

- Animated character that chases your shawarma cursor when you tap `Feed`
- Button grid with retro chrome styling straight out of 2003
- Randomized “ship it” style phrases whenever the character eats
- Built with React, Vite, Tailwind, and Framer Motion—no backend required
